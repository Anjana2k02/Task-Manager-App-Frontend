# TaskMaster AI – Task Management with Facial Expression Monitoring

## 📝 Project Overview

**TaskMaster AI** is an intelligent task management system designed as part of our 3rd Year 2nd Semester **ITPM (Information Technology Project Management)** module at **SLIIT**. This platform enhances traditional task management features with real-time **facial expression recognition**, enabling administrators and managers to monitor users’ emotional states while working.

The facial expression system provides an additional layer of insight to support better decision-making in task delegation, workload balancing, and team wellbeing.

---

## 🚀 Key Features

- ✅ User registration and role-based login (Admin, Manager, Employee)
- 📋 Task creation, assignment, tracking, and completion
- 🧠 Real-time facial expression detection using Python (OpenCV + Deep Learning)
- 📊 Dashboard with insights on employee emotional patterns
- 🔒 Secure user authentication
- 🗂 MongoDB-based task and user data storage

---

## 🧑‍💻 Technologies Used

### Frontend:
- **React.js** – Interactive and responsive UI

### Backend:
- **Spring Boot (Java)** – RESTful API service

### Database:
- **MongoDB** – NoSQL database for task and user management

### Facial Expression System:
- **Python** (OpenCV, TensorFlow/Keras, DeepFace or similar)
- Detects expressions like: Happy, Sad, Angry, Neutral, Surprised
